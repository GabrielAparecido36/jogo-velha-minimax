## Instalação:

### Requisitos:

- Node 8 ou superior
- Angular CLI

### Procedimento

Abra um terminal na pasta do projeto e digite o seguinte comando para instalar as dependências do projeto:

```sh
npm install
```

e posteriormente o seguinte comando para executar o serviço em si:

```sh
ng s
```
